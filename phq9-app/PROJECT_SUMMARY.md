## 🏗️ 项目架构

```
phq9-app/
├── src/
│   ├── components/          # 可复用组件
│   ├── data/                # 静态数据(量表定义)
│   ├── pages/               # 页面组件
│   ├── stores/              # 状态管理
│   ├── App.tsx              # 主应用组件
│   └── main.tsx             # 应用入口
├── package.json             # 项目依赖
├── vite.config.ts           # Vite配置
├── tailwind.config.js       # Tailwind配置
└── tsconfig.json            # TypeScript配置
```

## 🚀 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **状态管理**: Zustand
- **路由**: React Router
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **通知**: Sonner

## 📋 核心功能

### 1. 心理健康评估
- 支持11种专业心理健康量表
- 智能答题界面，自动跳转
- 实时进度显示
- 专业结果分析

### 2. 账户管理
- 用户注册和登录
- 个人资料管理
- 评估历史记录
- 数据本地存储

### 3. 用户体验
- 深色/浅色主题切换
- 响应式设计
- 打印功能
- 数据导出

## 🔒 隐私保护

- 所有数据在浏览器本地处理
- 不上传任何个人数据到服务器
- 用户可随时清除本地数据
- 符合隐私保护要求
