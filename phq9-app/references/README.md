# references 目录说明

本目录用于集中存放与本项目相关的同行评议论文 PDF、技术报告及其摘要，以支持：
- 学术可追溯性
- 实验复现与方法比对
- 项目文档中规范引用

建议结构：
- papers/：原始 PDF 或技术报告文件
- summaries/：每篇文献 1 页中文总结（研究目的、方法、主要结果、可应用技术点）
- misc/：补充材料（例如量表手册、附录、研究方案等）

> 注意：由于版权与访问许可限制，请由具备数据库访问权限的成员从 IEEE Xplore、ScienceDirect、SpringerLink、JMIR 等平台下载 PDF，并按照如下命名约定存放至 `papers/`：
> - 年份_第一作者姓_简要标题关键词.pdf
>   - 例如：`2024_Xu_iACT2.0_med_students.pdf`

与文献如何整合到项目、对照实验设计与引用关系，请参见仓库根目录的 `ACADEMIC_INTEGRATION.md`。
